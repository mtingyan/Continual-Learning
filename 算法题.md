# 字符串
题目描述：从字符串 A 中剔除字符串 B 中出现的所有字符
```python
def remove_chars(A, B):
    remove_set = set(B)  # 把B转为集合，加速查找
    result = ''.join([c for c in A if c not in remove_set])
    return result

# 示例
A = "abcdefg"
B = "bdf"
print(remove_chars(A, B))  # 输出 'aceg'

```
